    1. Add imports at top of the existing import block:
       ```python

       from mcp.types import ErrorData, INTERNAL_ERROR

       ```

    2. Add three private helpers ABOVE get_page (module-level functions, not decorated):

       `_count_blocks(blocks: list[BlockEntity]) -> int`
       — returns total block count including all descendants recursively

       `_dedup_children(children: list[BlockEntity], seen: set[str]) -> list[BlockEntity]`
       — for each child: skip if child.uuid is in seen, else add to seen and recurse into child.children; return filtered list

       `_parse_block_tree(raw_blocks: list) -> list[BlockEntity]`
       — iterate raw_blocks: call BlockEntity.model_validate(raw) for each, skip if block.uuid already in seen set, add uuid to seen, call _dedup_children on block.children with the SAME seen set, append to result

       Key: `seen` is a SINGLE set passed through the entire tree (not reset per top-level block). This is what prevents cross-subtree duplication.

    3. Add the @mcp.tool() async def get_page(ctx: Context, name: str) -> str:
       - Extract app_ctx and client from ctx.request_context.lifespan_context
       - Call client._call("logseq.Editor.getPage", name)
       - If result is None: raise McpError(ErrorData(code=INTERNAL_ERROR, message=f"page not found: {name}"))
       - Parse: page = PageEntity.model_validate(page_raw)
       - Call client._call("logseq.Editor.getPageBlocksTree", name)
       - Guard: blocks_raw = result if isinstance(result, list) else []
       - blocks = _parse_block_tree(blocks_raw)
       - Return json.dumps({"page": page.model_dump(by_alias=False), "blocks": [b.model_dump(by_alias=False) for b in blocks], "block_count": _count_blocks(blocks)})

    CRITICAL ANTI-PATTERN TO AVOID: Do NOT call _parse_block_tree or _dedup_children on children that are already embedded inside BlockEntity objects returned by model_validate — model_validate already recursively parses the entire children tree. The dedup pass only needs to check UUIDs, not re-parse the structure.
