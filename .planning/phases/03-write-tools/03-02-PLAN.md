Required implementation shape:
```python
from mcp import McpError
from mcp.server.fastmcp import Context
from logseq_mcp.server import mcp, AppContext
```

Primary RPCs for this plan:
```python
await client._call("logseq.Editor.createPage", name, properties, opts)
await client._call("logseq.Editor.appendBlockInPage", page_name, content, opts)
await client._call("logseq.Editor.insertBlock", parent_uuid, content, opts)
await client._call("logseq.Editor.getPage", page_name)
await client._call("logseq.Editor.getPageBlocksTree", page_name)
```

Input normalization contract:
- plain string -> block node with `content`, empty `properties`, empty `children`
- object -> validated recursive node with required `content`
- list -> normalized in caller order before any write RPC runs

Out of scope for this plan:
- `block_update`
- `block_delete`
- `delete_page`
- `rename_page`
- `move_block`
</interfaces>
